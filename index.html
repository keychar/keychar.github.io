浮点数格式

浮点数7.5如何存储

+7.5 = 3.75 * (21) = 1.875 * (22)
Sign = 0
Exp - 127 = 2
Exp = 129 = 1000 0001(Binary)
0.875 = 0.5 + 0.25 + 0.125
= (2-1) + (2-2) + (2-3)
Digit = 111 0000 0000 0000 0000 0000(Binray)
